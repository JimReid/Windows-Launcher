Windows Launcher
A lightweight, customizable application launcher for Windows that keeps your desktop clean and organized.

What It Does
Windows Launcher gives you a compact, flexible panel that sits anywhere on your screen and replaces the large shortcut icons cluttering your desktop. Instead of a messy desktop full of big icons, you get a tidy panel of small buttons — one for each app you want quick access to.

Features
Resizable and movable — drag it to any position on your screen and resize it to any shape you like
Remembers its position — after a reboot, the Launcher comes back exactly where you left it
Drag and drop shortcuts — simply drag a Windows shortcut onto the Launcher and a button is automatically created matching that shortcut's icon
Tooltips on every button — hover over any button to see the name of the application
One-click launching — left-click any button to launch the app instantly
Easy removal — right-click any button to remove it from the Launcher
Reorderable buttons — drag buttons into any order you like; they arrange left to right, top to bottom
Auto-start on boot — add the Launcher to your Windows Startup folder so it's ready every time you boot up

How to Set Up Auto-Start
Press Windows + R to open the Run dialog
Type shell:startup and press Enter
Drag a shortcut to the Launcher into that folder
The Launcher will now start automatically every time Windows boots

How to Use
Download and run the Launcher
Drag any desktop shortcut onto the Launcher panel — a button will appear
Delete the original shortcut from your desktop to free up space
Left-click a button to launch the app
Right-click a button to remove it
Drag buttons to rearrange them in your preferred order

Tips
Position the Launcher on the side of your screen (left or right edge works great) to keep your main workspace clear
Resize the panel to control how many buttons appear per row — for example, 3 columns across with several rows makes a very compact and handy layout
Press F1 inside the Launcher for built-in help

Requirements
Windows 10 or Windows 11
.NET runtime may be required — if not already installed on your system, Windows will prompt you to download it automatically
